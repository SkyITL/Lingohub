# NACLO Problem Labeling Taxonomy

## Linguistic Subdisciplines

### Core Areas
- **phonology** - Sound patterns, phonemes, allophones, syllable structure
- **morphology** - Word formation, affixes, compounding, derivation
- **syntax** - Sentence structure, word order, grammatical relations
- **semantics** - Meaning, semantic roles, truth conditions
- **pragmatics** - Context, discourse, speech acts

### Specialized Areas
- **writing-systems** - Scripts, orthography, transliteration
- **historical-linguistics** - Language change, reconstruction, cognates
- **comparative-linguistics** - Language comparison, typology
- **computational** - Algorithms, formal grammars, encoding
- **sociolinguistics** - Language variation, dialects, registers

## Problem Types

### By Skill Required
- **pattern-recognition** - Finding regularities in data
- **rule-formulation** - Deriving linguistic rules
- **translation** - Translating between languages
- **reconstruction** - Reconstructing proto-forms or missing data
- **decipherment** - Understanding unknown writing systems
- **algorithm-design** - Creating computational solutions

### By Data Type
- **paradigm** - Morphological tables, conjugations, declensions
- **word-list** - Lists of words with meanings
- **sentence-pairs** - Parallel texts
- **glossed-text** - Interlinear glossing
- **script-data** - Writing system examples

## Difficulty Levels

- **beginner** - Basic pattern recognition, 1-2 rules
- **intermediate** - Multiple interacting rules, moderate complexity
- **advanced** - Complex systems, multiple layers of analysis
- **expert** - Requires deep linguistic knowledge or complex reasoning

## Special Features

### Cognitive Skills
- **logical-reasoning** - Deductive reasoning required
- **spatial-reasoning** - Visual/spatial thinking
- **algorithmic-thinking** - Computational approach needed
- **creative-problem-solving** - Non-obvious solutions

### Problem Characteristics
- **multiple-languages** - Involves 2+ languages
- **diachronic** - Historical change over time
- **synchronic** - Single time period
- **real-world-application** - Practical uses (translation, NLP, etc.)
- **puzzle** - Pure logic puzzle (Yoda-speak, constraints, etc.)
