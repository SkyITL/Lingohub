## Linguistic Insights

This problem tests:
1. **Total Reduplication**: Complete doubling of verb root
2. **Productive Derivation**: Systematic meaning changes
3. **Valency Changes**: How reduplication affects argument structure
4. **Person/Number Agreement**: Subject and object marking

## Key Patterns

### Reduplication Functions:
- Intensive/iterative meaning
- Valency changes
- Aspect modification

### Agreement:
- Subject person/number marking
- Object person/number marking
- Complex portmanteau morphemes
