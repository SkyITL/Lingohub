## Problem Description

Words with derivational suffixes and their English translations in random order.

### Tasks:
- (a) Determine correct correspondences (given: moo = eye)
- (b) Translate into English (given: toŋgotoŋgo = bow)
- (c) Translate into Koyra Chiini (given: baasu = well)

**(!!)** No additional explanation required

## Notes

- **j** = j in 'jam'
- **y** = y in 'yam'
- **ŋ** and **ɲ** are consonants
- Doubled letters denote length
- A **water recipient** is a rubber bag for drawing water from wells

## Linguistic Insights

This problem tests:
1. **Derivational Morphology**: Productive suffixes with specific semantic functions
2. **Semantic Composition**: Understanding how meanings combine
3. **Word Formation**: Systematic derivational processes
4. **Pattern Recognition**: Identifying suffix functions from limited data

## Key Patterns

### Derivational Suffixes:
- **-ije/-jje**: Forms derived nouns (containers, parts)
- **-korfo**: Forms agent nouns or related entities
- **-ney**: Forms abstract nouns (qualities, states)
- **-ɲa**: Possessive or relational suffix

### Semantic Functions:
- Container/part relationships (seed → pod)
- Abstract quality formation (hot → heat, cold → coldness)
- Agent/person nouns (town → townsperson)
- Age-related derivation (old age, aged)
