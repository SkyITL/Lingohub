# APLO Problems: Linguistic Skill Taxonomy

This document provides a comprehensive taxonomy of linguistic skills and tags for APLO problems, inspired by Luogu's problem tagging system.

## Skill Categories

### 1. Core Linguistic Subfields (核心语言学分支)

- **phonology** (音系学) - Sound patterns, stress, tone, phonological rules
- **morphology** (形态学) - Word formation, inflection, derivation, compounding
- **syntax** (句法学) - Sentence structure, word order, grammatical relations
- **semantics** (语义学) - Meaning, semantic roles
- **writing-systems** (文字系统) - Scripts, orthography, spelling rules

### 2. Specialized Topics (专门主题)

- **number-systems** (数字系统) - Numeral systems, mathematical expressions
- **kinship** (亲属关系) - Kinship terminology and systems
- **possession** (领属关系) - Possessive constructions
- **agreement** (一致关系) - Subject-verb agreement, noun-adjective agreement
- **case-marking** (格标记) - Nominative, accusative, ergative systems
- **voice-focus** (语态焦点) - Active, passive, applicative, focus systems
- **evidentiality** (证据性) - Evidential markers
- **tense-aspect** (时体) - Tense and aspect marking
- **person-number** (人称数) - Person and number marking systems
- **gender** (性) - Grammatical gender systems
- **tone** (声调) - Tonal systems and patterns
- **stress** (重音) - Stress patterns and rules
- **reduplication** (重叠) - Reduplication patterns
- **compounding** (复合) - Compound word formation
- **derivation** (派生) - Derivational morphology
- **loanwords** (借词) - Loanword adaptation
- **root-pattern** (词根模式) - Root-and-pattern morphology (Semitic)

### 3. Language Families (语系)

- **austronesian** (南岛语系)
- **pama-nyungan** (帕马-恩永甘语系)
- **sino-tibetan** (汉藏语系)
- **trans-new-guinea** (跨新几内亚语系)
- **afroasiatic** (亚非语系)
- **nilo-saharan** (尼罗-撒哈拉语系)
- **macro-je** (宏热语系)
- **arawakan** (阿拉瓦克语系)
- **arawan** (阿拉万语系)
- **uto-aztecan** (尤特-阿兹特克语系)
- **torricelli** (托里切利语系)
- **songhay** (桑海语系)
- **turkic** (突厥语系)
- **misumalpan** (米苏马尔潘语系)
- **language-isolate** (孤立语言)

### 4. Difficulty Levels (难度等级)

- **beginner** (入门) - Simple pattern recognition
- **intermediate** (中级) - Multi-step reasoning, multiple interacting patterns
- **advanced** (高级) - Complex interactions, abstract patterns, ambiguity

### 5. Problem-Solving Skills (解题技能)

- **pattern-recognition** (模式识别) - Identifying regular patterns
- **segmentation** (切分) - Breaking down words into morphemes
- **paradigm-building** (范式构建) - Building inflectional/derivational paradigms
- **constraint-satisfaction** (约束满足) - Multiple interacting constraints
- **ambiguity-resolution** (歧义消解) - Handling multiple possible analyses
- **logical-deduction** (逻辑推理) - Step-by-step logical reasoning

 Problem-by-Problem Analysis

### APLO 2024

#### Problem 1: Tutuba (20 points)
- **Language**: Tutuba (Austronesian, Oceanic)
- **Country**: Vanuatu
- **Core Skills**: morphology, possession, agreement
- **Specific Topics**: possessive suffixes, noun classes, cultural semantics
- **Difficulty**: intermediate
- **Key Features**: Different possession patterns for different noun types; one animal treated as property

#### Problem 2: Wemba Wemba (20 points)
- **Language**: Wemba Wemba (Pama-Nyungan, Kulinic) - EXTINCT
- **Country**: Australia
- **Core Skills**: morphology, compounding, semantics
- **Specific Topics**: body-part terms, semantic extension, compounding
- **Difficulty**: intermediate
- **Key Features**: Body parts used metaphorically for other concepts (head → roof, eye → door, etc.)

#### Problem 3: Bambara (20 points)
- **Language**: Bambara (Mande)
- **Country**: Mali
- **Core Skills**: phonology, loanwords
- **Specific Topics**: loanword adaptation, phonological rules, syllable structure
- **Difficulty**: intermediate
- **Key Features**: French loanwords adapted to Bambara phonology; epenthetic vowels, cluster resolution

#### Problem 4: Mairasi (20 points)
- **Language**: Mairasi (Mairasi family)
- **Country**: Indonesia (West Papua)
- **Core Skills**: morphology, syntax, agreement
- **Specific Topics**: verb agreement, number marking (singular/plural), reduplication
- **Difficulty**: intermediate-advanced
- **Key Features**: Complex verb morphology with subject/object agreement and aspect

#### Problem 5: Chungli Ao (20 points)
- **Language**: Chungli Ao (Sino-Tibetan)
- **Country**: India (Nagaland)
- **Core Skills**: number-systems
- **Specific Topics**: base-20 numeral system, arithmetic
- **Difficulty**: intermediate
