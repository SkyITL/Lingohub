## Problem Description

Four representations of 97:
- Somali (Latin): toddoba iyo sagaashan
- Somali (Osmanya): [Osmanya script]
- Osmanya numerals: [symbols]
- Arabic numerals: 97

10 arithmetic equalities with blanks to fill.

### Tasks:
- (a) Fill blanks A-D in Osmanya numerals
- (b) Write equalities 1-9 in Arabic numerals
- (c) Write in Somali Latin script
- (d) Write in Somali Osmanya script

## Notes

- **(!)** No additional explanation required
- The Osmanya script was created 1920-1922
- Recognized as official script in 1961
- Now mostly replaced by Latin script

## Linguistic Insights

This problem tests:
1. **Base-20 Number System**: Vigesimal system understanding
2. **Alternative Writing System**: Osmanya script decipherment
3. **Number Word Formation**: Compositional semantics
4. **Cross-Script Translation**: Converting between writing systems
