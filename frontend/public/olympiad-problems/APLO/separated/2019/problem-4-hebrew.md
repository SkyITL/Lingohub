## Problem Description

Hebrew verb forms with their English translations in arbitrary order, demonstrating the Semitic root-and-pattern morphology system (binyanim).

### Tasks:
- (a) Determine the correct correspondence (14 pairs)
- (b) Translate 4 Hebrew forms into English
- (c) Translate 3 English sentences into Hebrew

## Notes

- **ə** ≈ o in "worm"
- **kh**, **sh** and **q** are consonants
- Masculine and feminine 2nd person forms are distinguished
- Various verbal stems (binyanim) with different meanings

## Linguistic Insights

This problem tests:
1. **Root-and-Pattern Morphology**: Semitic nonconcatenative system
2. **Consonantal Roots**: Typically 3-consonant roots carrying core meaning
3. **Binyanim System**: Vowel patterns and affixes modify root meaning
4. **Derivational Relationships**: Same root in different patterns = related meanings
5. **Person-Gender-Number Agreement**: Verb forms mark these features

## Key Patterns

### Nonconcatenative Morphology:
- **Roots**: Consonantal skeletons (e.g., k-t-v "write", g-d-l "grow", etc.)
- **Patterns**: Vowel templates and affixes interleave with roots
- Not simple prefix+stem+suffix like most languages

### Binyanim (Verb Stems):
Traditional 7 binyanim patterns with different functions:
- **Pa'al** (basic): katəva "she wrote", gadəla "she grew up"
- **Pi'el** (intensive/repetitive): ginev "he stole repeatedly"
- **Hif'il** (causative): higdil "he grew (something)", higniva "she smuggled"
- **Hitpa'el** (reflexive/reciprocal): hitxatavt "you corresponded", hitbatel "he shamed himself"
- Each pattern has characteristic vowels and affixes

### Person-Gender-Number:
- **-ti**: 1st person singular "I"
- **-ta** (masc) / **-t** (fem): 2nd person singular "you"
- **Ø** or **-a**: 3rd person singular "he/she"
- Gender distinction in 2nd/3rd person

### Semantic Derivation:
- Same root, different binyan = related meanings
- k-t-v: katav "wrote" → kitavti "I rewrote" → hitxatavt "you corresponded"
- g-d-l: gadəla "grew up" → higdil "grew (trans)" → gadalti "I grew up"
- Patterns add causative, intensive, reflexive, reciprocal meanings

## Difficulty Analysis

**Level**: Intermediate-Advanced

**Reasoning**:
- Nonconcatenative morphology unfamiliar to most learners
- Root extraction requires identifying non-adjacent consonants
- Multiple interacting patterns (binyanim)
- Person, gender, and number marking overlaid on root-pattern system
- Semantic relationships between binyanim subtle
- Requires abstract thinking about morphological structure
- Iconic introduction to Semitic linguistics
