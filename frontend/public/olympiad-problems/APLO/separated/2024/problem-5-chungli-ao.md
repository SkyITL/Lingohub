## Problem Description

Here are some equalities written in the Chungli dialect of the Ao language:

1. **tenemri ka + metsv̥ri asv̥m = tenem ser metsv̥ri pezv̥**
2. **liri pezv̥ + semv̥r maben pungu = tenem ser metsv̥maben tv̥ko**
3. **tenet + ti = metsv̥maben pungu**
4. **metsv̥maben trok × pungu = lir anasv̥**
5. **teri ana × ana + rokv̥ri asv̥m = telang tv̥ko maben tenet**
6. **semv̥r maben ti + liri ana = tenem ser metsv̥**
7. **tv̥ko × trok = tenemri pezv̥**
8. **lir anasv̥ri pezv̥ − lir maben pungu = tenem maben tv̥ko**
9. **teri asv̥m × trok = lir anasv̥maben ti**

### Task (a)
Write the equalities (1–9) in digits, given that **metsv̥maben ti** is 18.

### Task (b)
Write in Chungli Ao: **10**, **31**, **36**, **58**, **93**

If an exact expression cannot be determined from the given data, write out all possibilities.

## Notes

- **v̥** is a vowel.

## Key Patterns to Discover

### Base-20 (Vigesimal) System:
- The system is based on 20 as a fundamental unit
- Numbers are expressed using multiples of 20 plus remainders
- Similar to systems found in some Celtic languages (e.g., French "quatre-vingts" = 4×20 = 80)

### Numeral Morphology:
- **Basic numerals**: Small numbers (1-20)
- **Compositional patterns**: How numbers combine
  - **maben** likely means "and" or serves as a connector
  - **ri** appears to be a suffix/particle
  - **ser** may indicate a specific arithmetic relationship

### Arithmetic Clues:
- Given: **metsv̥maben ti = 18**
- This suggests:
  - **metsv̥** and **ti** are component numerals
  - **maben** is a combining element
  - Need to deduce which numbers they represent

### Discovery Strategy:
1. Use equation (3) to find values: **tenet + ti = metsv̥maben pungu**
2. Use equation (7) to establish relationships: **tv̥ko × trok = tenemri pezv̥**
3. Build up the numeral system systematically
4. Identify patterns for:
   - Base numbers (1-5)
   - Mid-range numbers (6-10)
   - Teens (11-19)
   - Twenties and beyond

### Expected Patterns:
- **-ri**: Possible multiplicative or modificational suffix
- **maben**: Additive conjunction ("and")
- **ser**: Possible multiplicative marker or separator
- **Multiples of 20**: Special forms for 20, 40, 60, 80, etc.
