## Problem Description

A table containing French verbs with prefixes (left column) and the corresponding verbs without prefixes (right column), along with English translations of all. The shaded cells mean that there is a prefixed verb there with no prefixless counterpart. In some verbs the prefixes have been left out.

### Tasks:
- **Assignment**: Fill in the gaps using information from the table. Explain your solution.

## Notes

- French prefixes include: **re-**, **dé-**, **r-**, etc.
- Some prefixed verbs have no corresponding base form (shaded cells)
- Prefixes can have multiple meanings or functions
- Problem tests understanding of French derivational morphology

## Linguistic Insights

This problem tests:
1. **Prefix Semantics**: Understanding meanings of French prefixes (re-, dé-, etc.)
2. **Productivity**: Which bases can take which prefixes
3. **Semantic Compositionality**: How prefix + base = derived meaning
4. **Lexical Gaps**: Some prefixed forms exist without bases (or vice versa)
5. **Morphophonology**: Prefix allomorphy (re- vs. r- vs. ré-)

## Key Patterns

### French Prefixes:

**re-/r-/ré-** (repetition, "again"):
- réagir "react"
- recommencer "recommence/begin again"
- recomposer "compose anew"
- réconcilier "reconcile"
- réconforter "comfort"
- recréer "recreate"
- récréer "amuse"
- redire "say again"
- réduire "reduce"
- rééditer "publish again"
- refaire "redo, remake"
- reformer "reform"
- former "form again"
- réfuter "refute"
- réincarner "reincarnate"
- rejouer "resume playing"
- relancer "throw again"
- remunérer "remunerate"
- rénover "renovate"
- réopérer "operate again"
- repartir "depart once more"
- répéter "repeat"
- résonner "sound"
- révéler "reveal"

**dé-/d-** (reversal, removal, "un-"):
- Many verbs with reversative meaning

### Allomorphy:
- **re-** before consonants
- **r-** before vowels (often)
- **ré-** with stress/emphasis or specific phonological contexts

### Semantic Patterns:
- Repetition: "do again"
- Reversal: "undo"
- Intensification: some cases
- Lexicalized meanings: meanings drifted from compositional
