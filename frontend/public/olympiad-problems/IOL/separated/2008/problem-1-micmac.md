## Notes
- [@] ≈ o in "abbot"
- [c] = ch in "church"
- [j] = j in "judge"
- [x] = ch in Scottish "loch"
- [ɣ] is the same sound but voiced
- [w] shows that the preceding consonant is pronounced with rounded lips
- The mark ¯ denotes vowel length

## Linguistic Insights

### Key Patterns to Discover

1. **Schwa Insertion/Deletion Pattern**:
   - Schwa [@] appears in phonetic transcription but not always in orthography
   - Initial consonant clusters in orthography get schwa before them: `tm` → [d@m], `gj` → [@kc]
   - Medial clusters also trigger schwa: `mgum` → [@mkum]

2. **Consonant Correspondences**:
   - `t` at word start → [d] (voicing)
   - `g` before front vowels → [k] or [@k]
   - `j` → [c] (affricate)
   - `q` → [x] (velar fricative)
   - `'` (apostrophe) marks glottal features or syllable boundaries

3. **Vowel Length Marking**:
   - Apostrophe `'` before vowel often indicates length: `i'` → [ī]
   - Not all long vowels have apostrophes in orthography

4. **Orthographic Conventions**:
   - Complex consonant clusters written without vowels
   - `e'` can represent [€] or [ē]
   - Final consonants: `g` → [k], `j` → [c]

### Solution Strategy

1. **Build Correspondence Tables**:
   - Map each orthographic symbol to its phonetic realization
   - Note positional variants (initial, medial, final)

2. **Identify Schwa Rules**:
   - Schwa appears before initial consonant clusters
   - Schwa appears between certain consonant combinations
   - Three-consonant clusters get schwa: `tpl` → [@tp@l]

3. **Apply Systematically**:
   - For Part A: Apply schwa insertion rules and consonant correspondences
   - For Part B: Reverse the process, removing predictable schwas

## Historical Context

The Listuguj orthography is a practical writing system developed for the Mi'kmaq language, balancing phonetic accuracy with writability. Unlike IPA, it uses familiar Roman letters and marks distinctive features (like length) with simple diacritics.
