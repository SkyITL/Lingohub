## Problem

Two languages, same region, same data structure - find the semantic patterns!

**Drehu words**: drai-hmitrötr, gaa-hmitrötr, i-drai, i-jun, i-wahnawa, jun, ngöne-gejë, ngöne-uma, nyine-thin, uma-hmitrötr

**English**: sanctuary, bunch of bananas, calendar, bone, church, coast, awl, Sunday, skeleton, wall

**Cemuhî words**: a-pulut, ba-bwén, ba-jié, bé-ôdu, bé-tii, bé-wöli, bé-wöli-wöta, tii, wöta

**English**: bed, animal, fork, cup, pencil, coast, to write, twilight, spur

**Known translations**:
- Drehu *gaa* → Cemuhî *a*
- Drehu *ngöne-gejë* → Cemuhî *ba-jié*
- Drehu *nyine* → Cemuhî *bé*
- Drehu *thin* → Cemuhî *wöli*

Tasks:
a) Match all words correctly
b) Deduce meanings of *wahnawa*, *drai* (Drehu) and *wöli*, *pulut* (Cemuhî)
c) Translate *i-bii* and *tusi-hmitrötr* (given: *tusi*='book', *bii*='bee')

## Key Insights
- Both languages use **compounding** extensively
- Prefixes mark noun classes or relationships
- *hmitrötr* means 'house/building' → church = God's house
- *gejë* = 'God', *uma* = 'mother' → Sunday = God's day
- Cross-linguistic cognates: *thin*/*wöli* both mean 'to write'

## Solution Approach
1. Identify compound parts
2. Find recurring morphemes
3. Build semantic field for "house", "day", "bone"
4. Use known translations as anchors
5. Apply compositionality principle
