## Problem Structure

**Data provided**: 18 Zoque phrases with English translations, showing patterns like:
- *nakpat* (a cactus)
- *mis nakpatpit* (with your cactus)
- *mokpittih* (only with the corn)
- *pokskuky2sm2taPm* (above the chairs)

**Tasks**:
a) Translate 6 Zoque phrases to English
b) Translate 6 English phrases to Zoque

## Key Patterns
- **Possessive prefixes**: *P2s n-* (my), *mis n-* (your)
- **Locative suffixes**: *-ky2sm2* (above), *-pit* (with), *-taPm* (plural marker)
- **Adverbial suffixes**: *-tih* (only), *-ˇseh* (as if/like)
- **Number marking**: Complex plural system

## Morpheme Analysis
- *P2s* = my (1sg possessive)
- *mis* = your (2sg possessive)
- *-pit* = with (comitative)
- *-ky2sm2* = above (superessive)
- *-taPm* = plural marker
- *-tih* = only (restrictive)
- *-ˇseh* = like/as if (similative)
- *-otoya* = for (benefactive)

## Solution Strategy
1. Segment each word into root + affixes
2. Build morpheme inventory
3. Identify allomorphy patterns
4. Note affix ordering (possessive + root + location/manner + number)
5. Apply systematically

## Difficulty: ⭐⭐⭐
- Polysynthetic/agglutinative structure
- Multiple affix positions
- Allomorphic variation
- Complex morphophonology (especially *2* vowel)
