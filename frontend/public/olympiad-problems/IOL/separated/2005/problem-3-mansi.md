## Problem Description

Consider the following Mansi numerals (transcribed in Roman letters):

### Given Numerals:
- 8: **ñollow**
- 15: **atxujplow**
- 49: **atlow nopɨl ontɨllow**
- 50: **atlow**
- 99: **ontɨlsāt ontɨllow**
- 555: **xōtsātn xūtlow nopɨl at**
- 900: **ontɨllowsāt**
- 918: **ontɨllowsāt ñollowxujplow**

### Tasks:
- **Assignment 1**: Determine the values of the following Mansi numerals:
  - **atsātn at**
  - **nolsāt nopɨl xōt**
  - **ontɨllowsātn ontɨllowxujplow**

- **Assignment 2**: Spell out the following numerals in Mansi: 58, 80, 716.

## Notes

- **ñ** is a specific consonant of the Mansi language
- **ɨ** is a specific vowel of the Mansi language
- A bar above a vowel indicates length (e.g., **ā**, **ō**, **ū**)

## Key Patterns

### Basic Numbers (deduced from examples):
- **1**: at (from "nopɨl **at**" = 555, where at = 1 × 100⁰)
- **8**: ñollow
- **10**: low (from at-xuj-p-**low** = 15)
- **50**: atlow
- **99**: ontɨlsāt ontɨllow
- **100**: sāt (from xōt-**sātn** = 600)

### Formation Patterns:
Based on the given data:
1. **Teens (11-19)**: Number + xujplow (e.g., 15 = atxujplow)
2. **Tens**: Various forms (-low suffix appears)
3. **Hundreds**: Number + sāt/sātn
4. **Compound Numbers**: Hundreds + tens + ones

### Morphological Components:
- **-low**: appears in multiples of 10
- **-sāt/-sātn**: appears in hundreds
- **xujplow**: appears in numbers 10-19
- **nopɨl**: conjunction "and" or compositional marker
- **ont-**: prefix appearing in 9, 90, 900

### Number System Structure:
Mansi uses a decimal (base-10) system with:
- Special forms for 1-10
- Compositional structure for larger numbers
- Morphological marking for hundreds and tens
- Possible subtractive or additive patterns
